##  Project Overview
This project is designed as a **Digital Literacy Portfolio**, where I take on the role of a *Student Digital Ambassador*. The aim is to explore and demonstrate essential digital skills required in today’s academic and professional environment.

Through this project, I have worked on multiple tasks such as creating awareness content, building a professional online presence, understanding collaboration tools, practising email etiquette, and learning about cyber safety. Each task reflects practical knowledge that can be applied in real-life situations.
